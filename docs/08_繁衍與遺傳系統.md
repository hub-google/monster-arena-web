# 八、 繁衍與遺傳系統 (Breeding & Genetics System)

交配繁衍是本遊戲維持活躍度與創造更強大怪獸的核心機制。當怪獸達到「成熟期」或以上階段，玩家即可與自己倉庫內的其他怪獸，或與好友的怪獸進行交配。交配後會誕生一顆新的「數位蛋」，並繼承親代的家族、屬性傾向與數值基因。

## 8.1 基因鑲嵌與「被動天賦 (Traits)」遺傳
除了基礎數值與屬性，每隻怪獸在出生時皆有機率自帶 1~2 個隨機「天賦（Traits）」（例如：[大胃王]、[夜貓子]、[吸血] 等）。
交配時，子代有極高機率繼承父母雙方所擁有的天賦。天賦系統的引入，賦予了相同數值怪獸不同的戰略與養成體驗，玩家將會為了追求「滿 IV + 完美雙天賦」進行反覆繁衍篩選。

## 8.2 家族遺傳規則 (Family Inheritance)
遊戲中共分為 7 大常規家族（龍、獸、惡魔、機械、植物、水棲、鳥）。子代蛋所屬的「家族」將決定牠未來的進化樹路線，其機率由父母雙方的家族決定：

*   **同家族交配**（例：龍族 + 龍族）：
    *   **100% 機率** 生出該家族的蛋。這也是純化基因最穩定的方式。
*   **跨家族交配**（例：龍族 + 獸族）：
    *   **45% 機率** 繼承父親（發起交配方）的家族。
    *   **45% 機率** 繼承母親（接受交配方）的家族。
    *   **10% 機率 基因突變**：生出一顆與父母完全無關的隨機家族蛋（在剩餘的 5 個家族中隨機抽選）。此機制讓玩家有機會在不具備某家族的情況下「抽」出新家族。

## 8.3 初始屬性遺傳規則 (Attribute Inheritance)
子代的**初始屬性**（決定幼年期進化到成長期時是哪一條路線，如：亞古獸/積木亞古獸/黑亞古獸）取決於父母雙方的屬性相剋邏輯。但須注意，本遊戲採用「動態屬性」，後續進化仍可依照玩家的培育方式發生轉變。

*   **同屬性交配**：
    *   疫苗 + 疫苗 = 100% 機率初始為【疫苗種】路線
    *   資料 + 資料 = 100% 機率初始為【資料種】路線
    *   病毒 + 病毒 = 100% 機率初始為【病毒種】路線
*   **相剋屬性交配**：(遵循 疫苗 剋 病毒 剋 資料 剋 疫苗 的循環，強勢方 70% / 突變第三方 30%)
    *   疫苗(剋) + 病毒(被剋) $\rightarrow$ 70% 疫苗路線，30% 資料路線
    *   病毒(剋) + 資料(被剋) $\rightarrow$ 70% 病毒路線，30% 疫苗路線
    *   資料(剋) + 疫苗(被剋) $\rightarrow$ 70% 資料路線，30% 病毒路線

## 8.4 戰鬥數值基因遺傳 (IV 個體值系統)
為了防止數值無限膨脹破壞生態系統，怪獸的最終基礎戰鬥數值並非無限疊加，而是由嚴格的公式組成，其中的變數為「基因個體係數 (IV)」。
*   最終數值公式：$S_{combat} = 階段標準值 \times 家族乘數 \times (1 + 基因個體係數)$
*   **IV 係數上限**：基因個體係數的絕對上限鎖死為 **30%** (即 0.30)。
*   **遺傳計算**：
    *   子代初始 IV 會繼承父母雙方的平均值：$IV_{base} = (IV_{p1} + IV_{p2}) / 2$
    *   在此基礎上進行浮動突變，但無論如何突變，最終 $IV_{child}$ 都不會超過 30%：
        *   **正常遺傳 (94% 機率)**：$IV_{child} = IV_{base} \pm 5\%$
        *   **黃金突變 (5% 機率)**：$IV_{child} = IV_{base} + 15\%$ (不可超越上限)
        *   **超黃金突變 (1% 機率)**：$IV_{child} = 30\%$ (必定滿個體值封頂)
*   這代表繁衍篩選只是為了逼近這 30% 的極限，而無法無限疊加突破天花板。滿 IV 的怪獸保證了數值的極致，但後續的比拼將著重於陣容策略與技能搭配。

## 8.5 隱藏條件觸發（特殊變異體）
*   若觸發「1% 超黃金突變」，且**父母雙方皆為特定組合的究極體**（例如：戰鬥暴龍獸 + 鋼鐵加魯魯獸），該子代蛋將不會孵化成常規家族，而是直接進入並解鎖隱藏的「家族 8：特殊變異體」（如：皇家騎士-奧米加獸 或 七大魔王）。
*   特殊變異體必定自帶滿 IV (30%) 且擁有極其強力的專屬天賦。
